# resources

## A

- **[Aseprite](https://aseprite.com)** (paid) ― an amazing (multi-platform) pixel art editor

## C

- **[CDNFonts](https://cdnfonts.com)** ― yet another font site *(some fonts on this site are NOT free, but can be used for personal stuff)*
- **[CSS-Tricks](https://css-tricks.com)** ― cool css snippets and guides

## D

- **[deploy2nekoweb](https://deploy.nekoweb.org/)** ― the most popular nekoweb deploy script

## F

- **[Figma](https://figma.com)** ― a whiteboard and design program
- **[Fontshare](https://fontshare.com/)** ― fonts. fonts. fonts.
- **[Fontworks archive](https://archive.org/download/Fontworks/)** ― very cool jp and eng fonts (none of these are "free")

## G

- **[Google Fonts](https://fonts.google.com)** ― has a bunch of free-to-use fonts and icons

## M

- **[MDN Web Docs](https://developer.mozilla.org/en-US/)** ― good for web dev documentation

## N

- **[nekoweb domain tutorial](https://domain.joo.sh)**
- **[nekoweb stats](https://maxpixels.moe/resources/nekoweb-stats/)**
- **[Neovim](https://neovim.io)** ― my favorite text editor, very customizable

## P

- **[pmoring](https://github.com/ThinLiquid/pmoring)** ― a highly efficient, and easy-to-use webring shell
- **[Photopea](https://photopea.com)** ― a photoshop alternative for the web
- **[Programming Fonts](https://www.programmingfonts.org/)** ― a place to test monospace fonts

## U

- **[Unsplash](https://unsplash.com)** ― a really big image library

## W

- **[w3schools](https://www.w3schools.com)** ― good for html snippets, great for beginners
